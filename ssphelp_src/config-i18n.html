設定：国際化

このページでは右クリックメニューの「設定」サブメニューにある項目「本体設定」で開くダイアログの、「国際化」ページについて解説しています。

このページでは国際化に関係する設定を行います。

表示の国際化に関する設定です。

各項目の解説

メニューフォント設定上書き / バルーンフォント設定上書き

メニューやバルーンで使うフォントを変更します。
日本語版Windows以外で使用する場合の文字化けの軽減に役立つほか、単に別のフォントで表示したいという場合にも使えるかと思います。

選択
フォント選択ダイアログを開き、表示に使うフォントを選びます。
標準設定
標準設定に戻します。

省略時の文字コード

日本語OS以外の環境で日本語ゴーストを起動する時などで、文字コードがらみの問題が起きて文字化けが発生した場合に変更します。
通常はすべてDefaultの状態で使用してください。SSP内部でなるべく自動認識します。

テキスト定義
descript.txt, install.txt, surfaces.txt等のテキスト定義ファイルの文字コードを指定します。
File I/O
ネットワーク更新、Zip解凍等のファイル入出力処理での文字コードを指定します。
readme
readme.txtファイルの文字コードを指定します。
SHIORI
SHIORI（AI）との通信で使用する文字コードを指定します。
MAKOTO
MAKOTOプラグインとの通信で使用する文字コードを指定します。
PIRO（OLD）
旧PIROプラグインとの通信で使用する文字コードを指定します。
SCRIPT（OLD）
旧スクリプト処理で使用する文字コードを指定します。

下部のボタン

ヘルプ
本体設定ダイアログの、設定中のページのヘルプ（つまりこのページ）を開きます。
ダイアログ右上の「？」マークも同様です。
閉じる
本体設定ダイアログを閉じます。
ダイアログ右上の「×」マークも同様です。
